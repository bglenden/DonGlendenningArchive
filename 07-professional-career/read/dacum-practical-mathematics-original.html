PRACTICAL MATHEMATICS*


A. CALCULATE USING WHOLE NUMBERS

 1.  Interpret the real system of numbers

 2. Read and write whole numbers

 3. Add whole numbers

 4. Subtract whole numbers

 5. Multiply whole numbers

 6. Divide whole numbers

 7. Solve whole numbers using order of operations

 8. Round off whole numbers

 9. Solve written problems using whole numbers

 10. Identify and apply prime and composite numbers


B. CALCULATE USING COMMON FRACTIONS

1. Read and write common fractions

2. Determine highest common factors

3. Reduce common fractions

4. Determine lowest common denominators

5. Arrange common fractions in order of size

6. Add common fractions

7. Subtract common fractions

8. Multiply common fractions

9. Divide common fractions

10. Solve common fractions using order of operations

11. Solve written problems using common fractions

12. Calculate using ratios and proportions


C. CALCULATE USING DECIMAL FRACTIONS

1. Read and write decimal fractions

2. Round off decimal fractions

3. Add decimal fractions

4. Subtract decimal fractions

5. Multiply decimal fractions

6. Divide decimal fractions 

7. Solve decimal fractions using order of operations

8. Arrange decimal fractions in order of size. 

9. Change decimal fractions to common fractions

10. Change common fractions to decimal fractions

11. Solve written problems using decimal fractions

12. Solve whole numbers, common fractions, and decimal fractions using order of operations

13. Square numbers

14. Determine square roots


D. CALCULATE USING PERCENTAGES

1. Change percentages to decimal fractions and vice-versa

2. Change percentages to common fractions and vice versa

3. Find the percent of a number

4. Find what percent one number is of another

5. Find the number when a percent of it is given

6. Solve written problems using percentages

7. Determine precision and accuracy of approximate numbers


E. APPLY PLANE GEOMETRY

1. Identify and use geometric instruments

2. Identify plane geometric shapes and their trigger parts 

3. Identify, construct, and measure angles

4. Identify, construct and measure line segments

5. Construct parallel and perpendicular line segments

6. Draw line segments in scale

7. Divide line segments in proportions

8. Identify and construct triangles and measure parts

9. Identify and construct quadrilaterals and measure parts 

10. Identify and construct regular polygons of more than four (4) sides

11. Construct circles

12. Bisect angles

13. Construct ellipses 

14. Construct hyperbolas and parabolas

15. Construct tangents to circles

16. Compute perimeters and circumferences

17. Calculate using the Pythagorean Theorem

18. Compute areas of geometric shapes

19. Compute volumes of geometric solids

20. Compute total and lateral areas of the geometric solids 

21. Compute geometric shapes involving loci 

22. Identify and apply terms associated with circles, ellipses, hyperbolas, and parabolas


F. SOLVE EQUATIONS AND INEQUALITIES

1. Calculate using positive, negative, and absolute numbers

2. Interpret algebraic equations and symbols

3. Perform basic operations on monomials

4. Perform basic operations on polynomials

5. Solve linear equations with one variable

6. Apply linear equations with one variable to word problems

7. Interpret and construct graphs

8. Solve simultaneous equations

9. Locate points on the rectangular and polar coordinate system 

10. Solve word problems involving systems of equations.

11. Solve problems by transposing formulas

12. Perform factoring operations

13. Solve problems involving factoring

14. Calculate using surds

15. Solve quadratic equations

16. Calculate using exponents

17. Calculate using logarithms

18. Calculate using direct and inverse variation

19. Solve problems involving progression

20. Solve equations involving inequalities


G. APPLY BASIC TRIGONOMETRIC FUNTIONS

1. Calculate using similar triangles

2. Identify and evaluate the six (6) trigonometric functions in a right triangle

3. Determine trigonometric functions using tables

4. Construct graphs of trigonometric functions

5. Solve problems using basic trigonometric functions

6. Calculate using angles of depression and elevation

7. Solve and apply trigonometric identities

  8. Solve problems involving angles greater than 90 degrees

9. Solve trigonometric problems using the laws of sine and cosine 

10. Convert degrees to radians and vice versa


H. CALCULATE USING VECTORS

1. Add and subtract vector quantities

2. Define and draw vector quantities 

3. Operate on vectors by scalar

4. Solve written problems by using vectors

5. Solve problems using “J” operator

6. Convert complex numbers from rectangular forms to polar and exponential forms

7. Calculate components of vectors to find resultants


I. APPLY MATHEMATICS TO LIFE SKILLS

1. Read and interpret word problems

2. Interpret and apply S.I system 

3. Interpret and convert units of measurement

4. Operate calculators

5. Operate computer terminals

6. Apply short methods in mathematical operations

7. Calculate averages 

8. Interpret and construct statistical tables and graphs 

9. Apply probability theory

10. Perform personal banking procedures

11. Interpret regulations and complete income tax forms

12. Read power meters and calculate consumption 

13. Assess inflation factors

14. Prepare budgets

15. Evaluate money agencies for dollar value

16. Interpret financial tables

17. Prepare profit and loss statements

18. Calculate interest

19. Determine and depreciation and appreciation 

20. Take inventory

21. Calculate payroll and submit deductions

22. Prepare invoices

23. Estimate materials, supplies and labour costs

24. Calculate unit pricing

25. Read and interpret utility bills


 






*Transcribed from a DACUM analysis prepared by Holland College ( Dated: February 1978)
