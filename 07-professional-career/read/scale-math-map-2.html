Citizen Math rating scale assessment tool with performance levels 1-6 specifically designed for mathematics instruction and evaluation. Describes progression from tasks requiring constant supervision (Level 2) through satisfactory independent completion (Levels 3-4) to high-level performance with initiative and adaptability (Levels 5-6). Includes fields for student name, instructor name, and date, making it a practical classroom assessment instrument. Notes specify ratings should be community performance standards-based. This concrete application reflects Glendenning's pedagogical focus on practical assessment tools that educators can implement in actual classroom settings.
