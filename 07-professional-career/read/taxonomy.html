Reference document presenting two taxonomies of cognitive learning levels. Bloom's taxonomy lists six levels: awareness, understanding/comprehension, application, analysis, synthesis, evaluation. Application taxonomy shows five levels from basic knowledge through real-world application with unpredictable problems. Used in Don's standards framework for assessing complexity of learning objectives.
